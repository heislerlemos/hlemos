# Manual

Faça clone do projecto no gihub

```bash
  git clone git@github.com:heislerlemos/hlemos.git
```

Aceda no directorio

```bash
  cd /app
```

Instale as dependencies

```bash
  bundle install
  yarn install
```

Incio o banco de dados e o servidor

```bash
  rails db:create
  rails db:migrate
  rails db:seed
  rails server
```
